AI Genie
AI Genie is a full-stack MERN (MongoDB, Express.js, React, Node.js) web application.
///////////////////////////////////////////////////

Project Overview
AI Genie is designed to provide AI-powered assistance through a modern web interface. The project demonstrates full-stack development skills including database integration, REST API development, and responsive frontend design.

///////////////////////////////////////////////////
Tech Stack
Frontend: React.js with CSS
Backend: Node.js with Express.js
Database: MongoDB
Language: JavaScript


///////////////////////////////////////////////////
Installation
Prerequisites
Node.js installed on your system
MongoDB database (local or cloud)


///////////////////////////////////////////////////
Backend Setup
Navigate to backend folder:
cd backend
Install dependencies:
npm install
Create .env file and add:
MONGO_URI=your_mongodb_connection_string
PORT=5000
Start backend server:
npm start

///////////////////////////////////////////////////
Frontend Setup
Navigate to frontend folder:
cd frontend
Install dependencies:
text
npm install
Start frontend application:
npm start
The application will run on localhost:3000 (frontend) and localhost:5000 (backend).
